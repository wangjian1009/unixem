UNIXem - README
===============

Updated:    13th August 2010
See the introduction in the accompanying doc/html/1.9.1/index.html file.


UNIXem is UNIX Emulation for Win32 Library. The library currently provides implementations of the following UNIX APIs:

- chdir(), getcwd(), mkdir(), rmdir()
- dlopen(), dlclose(), dlsym(), dlerror()
- getpagesize()
- gettimeofday()
- link(), unlink()
- glob(), globfree()
- mmap(), munmap(), msync()
- opendir(), readdir(), rewinddir(), closedir()
- pathconf(), realpath()
- readv(), writev()

UNIXem forked from http://synesis.com.au/software/unixem.html
